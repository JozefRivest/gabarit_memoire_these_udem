# Gabarit pour les thèses et les mémoires à l'Université de Montréal

Le présent dépôt contient tous les fichiers nécessaires pour 
écrire sa thèse ou son mémoire avec [Quarto](https://quarto.org).
Le gabarit est prêt à être utilisé dès maintenant! Les utilisateurs 
n'ont qu'à modifier le fichier `_quarto.yml` afin d'insérer leur nom,
le titre de leur document, s'il s'agit d'une thèse de doctorat ou 
de maîtrise ainsi que le personel évaluateur. Le document est clair 
et simple, et il est annoté afin d'éviter tous malentendus. 

Pour utiliser le gabarit, les utilisateurs peuvent soit:

1. « Fork » le dépôt.
2. Cloner le dépôt.
3. Télécharger les fichiers en cliquant sur le bouton vert `code` 
et « Download zip ».
4. Utiliser l'outil terminal de Quarto suivant

```bash
quarto use template JozefRivest/gabarit_memoire_these_udem
```
